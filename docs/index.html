Get running in minutes

Three steps to
your local AI.

No configuration files to edit. No packages to install. Just download, flash, and boot.

STEP 01 / 03

Download

Grab the ISO from GitHub Releases. Choose Server (headless) or Desktop (full GUI). Verify the checksum.

# verify after download
sha256sum -c SHA256SUMS
ollamalinux-0.1.0-server-amd64.iso: OK

STEP 02 / 03

Flash

Write the ISO to a USB drive with dd on Linux/macOS, or use balenaEtcher on Windows.

# replace /dev/sdX with your USB drive
sudo dd if=ollamalinux-0.1.0-server-amd64.iso
    of=/dev/sdX bs=4M status=progress
    conv=fsync

STEP 03 / 03

Boot

Boot from the USB. The first-boot wizard launches automatically. Follow the prompts — your AI will be ready in minutes.

# after first-boot wizard completes
ollama run llama3.2
# web UI available at
http://127.0.0.1:8080

Before you download

System Requirements

OllamaLinux runs on any x86_64 machine. A GPU dramatically improves inference speed but is not required.

Component Minimum Recommended
CPU x86_64, 4 cores x86_64, 8+ cores
RAM 8 GB 16 GB+
Storage 50 GB 100 GB+
GPU None (CPU inference) NVIDIA RTX / AMD RX 6000+
Network Required (model downloads) Broadband

GPU drivers are downloaded and installed at first boot. An internet connection is required for the initial model download.
